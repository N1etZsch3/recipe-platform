# 社交与私信

本模块支持关注关系、私信聊天与会话列表。

## 1. 关注与粉丝

- `POST /api/v1/social/follow/{userId}`
  - 关注/取关切换。
- `GET /api/v1/social/follows`
  - 我关注的用户列表（分页）。
- `GET /api/v1/social/fans`
  - 我的粉丝列表（分页）。
- `GET /api/v1/social/user/{userId}`
  - 用户公开信息（含是否已关注）。

## 2. 私信与会话

- `POST /api/v1/social/messages`
  - 发送私信：`receiverId` / `content`。
  - 限制：非互关状态仅允许发送一条消息。
- `GET /api/v1/social/messages/{userId}`
  - 获取与指定用户的聊天记录（分页）。
- `GET /api/v1/social/conversations`
  - 获取会话列表（含未读数、最后消息、关注用户补全）。
- `PUT /api/v1/social/messages/read/{senderId}`
  - 将某个发送者的未读消息标记为已读。

## 3. 业务细节

- 关注时会向被关注者推送“新粉丝”通知。
- 发送私信会向对方推送“新私信”通知。
- 会话列表会合并“已产生聊天记录的用户”与“我关注的用户”。
